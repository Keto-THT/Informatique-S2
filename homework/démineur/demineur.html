Comment jouer?

Le but du jeu est de trouver les 10 bombes cachées dans la grille de 100 cases.

  • Chaque case peut être vide ou contenir une bombe. En cliquant sur une case, vous révélez son contenu.
  • Si vous cliquez sur une bombe, vous perdez la partie. Si vous cliquez sur une case vide, elle affichera un chiffre indiquant le nombre de bombes présentes dans les cases adjacentes.
  • Utilisez ces indices pour éviter les bombes et trouver toutes les cases vides pour gagner la partie.

Bonne chance!
